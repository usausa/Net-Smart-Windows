# Smart.Windows .NET - MVVM helper library for WPF

## What does if contain ?

* Basic converters.
* Observable commands.
* Actions, Behaviors and Triggers.
* Markup extensions.
* Messenger.
* Resolver(DI Container) integration.
* Base class for ViewModel.
